# GD-Texture-Splitter-Android

Aplicación Android offline para dividir, fusionar y convertir texturas de Geometry Dash usando PNG + PLIST (cocos2d-x).

## ✨ Características

* 🎨 UI Material 3 Expressive
* 📱 Navegación flotante (Split, Merge, Port, Settings)
* 🌍 Multidioma:

  * Español (México)
  * Inglés (EE.UU.)
  * Portugués (Brasil)
  * Japonés, Alemán, Ruso, Indonesio, Francés, Italiano, Coreano
* ⚙️ Funciona completamente offline

## 🧩 Funciones

### 🔪 Split

* Requiere PNG + PLIST
* Extrae sprites individuales
* Exporta como ZIP

### 🧩 Merge

* Crea Sprite Sheets
* Genera archivo PLIST automáticamente
* Permite presets o nombre manual

### 📉 Port

* Convierte texturas:

  * UHD
  * HD
  * LOW
* Genera PNG optimizado + PLIST

### ⚙️ Opciones avanzadas

* Alternar rotación de texturas
* Alternar recorte de sprites
* Ajustar spacing entre sprites

## 📂 Estructura de exportación

/storage/emulated/0/GD Texture Splitter/Export/

* Split/
* Merged/
* Ported/

## 🚀 Instalación

1. Clona el repositorio:

```bash
git clone https://github.com/PolarMinase33/gd-texture-splitter-android.git
```

2. Abre en Android Studio

3. Ejecuta en dispositivo Android (min SDK 26)
